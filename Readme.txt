Migration 

A partir d'angular 8 : 
Le compilateur Ivy reduit la taille des packages sur la taille des projet.
Ce qui reduit nettement la taille du proejt.

Il est important d'avoir la dernière version d'angular pour bénéficier des dernières fonctionnalités. 
On aura donc plus les bugs des versions précédentes. 

De plus la migration permet de résoudre des failles de sécurité.
